安装浏览器扩展插件 - CBX2 VPN Proxy

✅ 信息：

描述：CBX2 VPN Proxy by CB_X2_Jun. 中文发布版本.

支持的语言：简体中文（zh_CN）.

版本：v1.33.2.182

作者：CB_X2_Jun

⚠️ 重要提示：

由于浏览器安全限制，直接安装.crx文件会被禁用。建议使用下方的ZIP文件安装方法。

本扩展使用的代理列表：点击此处；本列表会不断由 CB_X2_Jun 手动更新，并添加新节点。由于是搜集的免费节点，很可能不稳定。如果有不可用的情况，欢迎到 GitHub Repo 提交 Issue。

Microsoft Edge - 微软 Edge 浏览器
Google Chrome - 谷歌 Chrome 浏览器

安装指引：

  1. 点击上方的 "下载ZIP / CRX文件" 按钮
  2. 在文件资源管理器找到下载的文件
  3. 打开Chrome，在地址栏输入: chrome://extensions/
  4. 开启右上角的 "开发者模式"
  5. 把ZIP / CRX文件拖拽进"扩展程序"页面
  6. 扩展即可正常使用
